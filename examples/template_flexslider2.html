jQuery FlexSlider 2 example. You need the FlexSlider 2 plugin from here for this example: http://flexslider.woothemes.com/thumbnail-slider.html Extract the zip file and copy it to your fileadmin-folder! Include the JS-file by Flexform or with TypoScript. Test it with this TypoScript: plugin.tx_karussell_pi1 { templateFile = EXT:karussell/examples/template_flexslider2.html jsFile = fileadmin/template/woothemes-FlexSlider/jquery.flexslider-min.js styleFile = fileadmin/template/woothemes-FlexSlider/flexslider.css jCarousel { dynamic = false } }
