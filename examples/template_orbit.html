Foundation-Orbit Slider-Template

You need the Foundation-Orbit-Slider from here: http://foundation.zurb.com/docs/orbit.php Copy the JavaScript and CSS from there to your fileadmin-folder!!! Test it with:

plugin.tx_karussell_pi1 {
  styleFile = fileadmin/template/foundation/stylesheets/foundation.css
  jsFile = fileadmin/template/foundation/javascripts/jquery.foundation.orbit.js
  templateFile = EXT:karussell/examples/template_orbit.html
}
