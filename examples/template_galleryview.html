jQuery GalleryView example. You need the GalleryView plugin from here for this example: http://spaceforaname.com/galleryview/index.html Extract the zip file and copy it to your fileadmin-folder! Include the JS-file by Flexform or with TypoScript like this: plugin.tx_karussell_pi1.jsFile = fileadmin/template/javascript/jquery.galleryview-3.0-dev.js Note: you need jquery.timers too! Copy that file into the galleryview-js-file or include it another way. Test it with this TypoScript: plugin.tx_karussell_pi1 { templateFile = EXT:karussell/examples/template_galleryview.html jsFile = fileadmin/template/GalleryView/js/jquery.galleryview-3.0-dev.js styleFile = fileadmin/template/GalleryView/css/jquery.galleryview-3.0-dev.css jCarousel { dynamic = false animation = 1100 start = 1 auto = false } } Note: you can define the start-element with a URL parameter: "&start=4" starts with element no. 4
